========================================
CORRA STUDIO - INSTALLATION GUIDE
========================================

SYSTEM REQUIREMENTS:
- Windows 10 or Windows 11 (64-bit)
- .NET 8 Runtime (included)
- 4GB RAM minimum (8GB recommended)
- 1GB free disk space
- Touchscreen (recommended for kiosk mode)

INSTALLATION STEPS:
1. Run CorraStudio-Setup.exe
2. Follow the installation wizard
3. Launch Corra Studio
4. Follow First Run Wizard to:
   - Activate license
   - Configure camera
   - Configure printer
   - Set up payment gateway

FIRST TIME SETUP:
1. Enter your license code
2. Connect your camera (DSLR or Webcam)
3. Connect your photo printer
4. Configure payment gateway (optional)

TROUBLESHOOTING:
- Check logs at: %LocalAppData%\CorraStudio\Logs
- Run as Administrator if permission issues
- Contact support@corrastudio.com for help
